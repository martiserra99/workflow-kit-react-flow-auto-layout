# React Flow Auto Layout

A standalone Node.js project with the `autoLayout` function that implements the layout algorithm.

For a full breakdown of how the algorithm works, read the article:  
👉 [React Flow Auto Layout: How to Automatically Position Nodes for Workflow Builders](https://www.workflowkit.app/blog/articles/react-flow-auto-layout)

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the project:

```bash
pnpm run start
```
